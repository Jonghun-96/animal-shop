🛒 React Shopping Mall

안녕하세요.
사용자 흐름과 구조 설계를 고민하는 프론트엔드 개발자 김종훈입니다.

이 프로젝트는 React와 TypeScript를 기반으로 제작한 쇼핑몰 웹 애플리케이션입니다.
상품 탐색부터 장바구니, 결제 흐름까지의 전반적인 사용자 경험을 직접 설계하고 구현했습니다.
