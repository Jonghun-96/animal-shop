# 🛒 React Shopping Mall

안녕하세요.
사용자 흐름과 상태 설계를 고민하는 프론트엔드 개발자 김종훈입니다.

이 프로젝트는 React와 TypeScript를 기반으로 제작한 쇼핑몰 웹 애플리케이션입니다.
상품 탐색부터 장바구니, 결제 완료 화면까지의 흐름을 직접 설계하고 구현했습니다.

## 🚀 Tech Stack
Frontend

React

TypeScript

Vite

State Management

Redux Toolkit

Routing

React Router

Styling

CSS (반응형 레이아웃, 다크모드)

## 📌 주요 기능

상품 목록 및 상세 페이지 구현

장바구니 담기 / 수량 변경 / 삭제 기능

Redux 기반 전역 상태 관리

로컬스토리지 연동을 통한 데이터 유지

React Router 기반 페이지 전환

반응형 UI 및 다크모드 지원

관리자 모드(Admin Mode) 구현
  
     관리자 전용 로그인 분리
  
     상품 등록 / 수정 / 삭제 기능 구현
  
     관리자 페이지 라우트 보호 (권한 기반 접근 제어)

## 🧠 데이터 처리 방식

본 프로젝트는 더미 데이터를 활용하여 상품 목록과 상세 정보를 구성했습니다.
외부 API 연동보다는 전역 상태 설계와 데이터 흐름 관리에 집중하여 구현했습니다.

장바구니 상태 변화, 수량 증감, 삭제 로직을 Redux를 통해 관리하며
예측 가능한 상태 구조를 설계하는 데 중점을 두었습니다.
